React UI Components Sample App





The purpose of this project is to get you familiarized with extending open-source React components and creating with them simple pages by using technologies like React / Flux / NPM / Webpack / Webpack Dev Server / Babel





The React DataTable and DropdownList components presented in these samples are extending Facebook's FixedDataTable and ReactWidgets's Dropdown List


The DataTable component supports the following use cases:
- lazy loading
- multiple selection
- async column sorting
- auto width
- percentage column widths
- async filtering
- loading progress
- no data custom message



The DropdownList component supports the following use case:
- disabled items



Disclaimer: these components are simple reference implementations and should not be used directly in production without further development!
